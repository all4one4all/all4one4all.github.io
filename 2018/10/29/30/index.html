网站设计与构建——CSS

发表于 2018-10-29 | 分类于 瞎记

CSS基础

为文档添加CSS样式

行内样式

写在 HTML 标签的 style 属性里。

嵌入样式

写在 HTML 的 head 元素中。

链接样式

把样式写在一个单独的文件里，即样式表。并 HTML 页面链接样式表。

在样式表里使用@import url(style.css)方式链接其他样式表， 且 @import 必须写在其他样式之前。

选择器

基本选择器

  • 通用选择器
    * {style}匹配文档中的所有元素
  • 类型选择器
    匹配某一类元素
    h1, h2 {style}会匹配所有<h1>与<h2>元素
  • 类选择器
    匹配指定的class属性的元素
    .box {style}会匹配所有 class 名为box的元素
  • ID选择器
    匹配指定的id属性的元素
    #box {style}会匹配 id 名为box的元素
  • 子元素选择器
    匹配指定元素的直接子元素
    p>a {style}会匹配所有父元素为<p>的<a>元素
  • 后代选择器
    匹配指定元素的子元素（不一定为直接子元素）
    p a {style}会匹配所有<p>中的<a>元素
  • 相邻同胞选择器
    匹配一个元素的相邻的兄弟元素
    span+a {style}会匹配元素后的第一个元素为<a>的<a>元素
  • 普通同胞选择器
    匹配一个元素的兄弟元素（不一定为相邻兄弟元素）
    span~a {style}会匹配<span>元素后的的<a>元素

属性选择器

  • 简单选择器
    []匹配一个指定属性的元素（与属性的值无关）
    input[type] {style}会匹配所有带有 type 属性的<input>元素
  • 精准选择器
    [=]匹配一个指定属性的元素，且该属性带有指定的值
    input[type="text"] {style}会匹配所有 type 为 text 的<input>元素
  • 开头选择器
    [^=]匹配一个指定属性的元素，且属性的值以某一指定字符串作为开头
    p[class^="box"] {style}会匹配所有 class 名开头为box的<p>元素
  • 包含选择器
    [*=]匹配一个指定属性的元素，且属性的值带有指定字符串
    p[class*="o"] {style}会匹配所有 class 名中带有o的<p>元素
  • 结尾选择器
    [$=]匹配一个指定属性的元素，且属性的值以某一指定字符串作为结尾
    p[class$="1"] {style}会匹配所有 class 名结尾为1的<p>元素
  • 部分选择器
    [~=]匹配一个指定属性的元素，且属性的值有以空格为间隔的某一指定字符串
    p[class~="dropdown"] {style}会匹配所有 class 名中带有dropdown且前后有空格间隔的多个名的<p>元素

伪类

UI伪类

  • :link :visited :hover :active
    链接伪类，链接会始终处于这四种状态的一种。分别为未被点击、此前点击过、鼠标悬停及正在点击
  • :focus
    点击时会获得聚焦
    input:focus {border:1px solid gray}会使<input>元素在点击输入时出现边框
  • :target
    点击一个指向页面中其他元素的链接，则那个元素即可用:target选中

结构伪类

  • :first-child
    选中一组同胞元素中的第一个元素
  • :last-child
    选中一组同胞元素中的最后一个元素
  • :nth-child(n)
    n可以为数字，即选中一组同胞元素中的第n个元素
    n可以为odd，即选中一组同胞元素中所有第奇数个元素
    n可以为even，即选中一组同胞元素中所有第偶数个元素

伪元素

  • ::first-letter
    选中文本段落的首字符
  • ::first-line
    选中文本段落的第一行。
  • ::before ::after
    在特定元素前面或后面添加特殊内容

继承

部分属性可以继承，即子元素继承父元素的属性。也可将属性值设为inherit来强制之大多数元素从父元素继承属性值。

层叠

多个CSS样式应用在同一元素上时其作用顺序即层叠样式。

  • 就近
    行内样式>嵌入样式>链接样式
  • 特指度
    具体的选择器>一般选择器
    设定样式>继承样式
  • 重要性
    !important;相较于其他样式权重最大
